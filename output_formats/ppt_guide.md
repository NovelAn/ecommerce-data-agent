# PPT 输出格式指南

## 适用场景

| 场景 | 推荐程度 | 说明 |
|------|----------|------|
| 高管汇报 | ⭐⭐⭐⭐⭐ | 最佳选择 |
| 周报/月报 | ⭐⭐⭐⭐⭐ | 最佳选择 |
| 季度复盘 | ⭐⭐⭐⭐⭐ | 最佳选择 |
| 会议演示 | ⭐⭐⭐⭐⭐ | 最佳选择 |
| 日常分析 | ⭐⭐ | HTML 更灵活 |

## PPT 结构模板

### 高管汇报 (10-15 页)
```
1. 封面
2. 执行摘要 (1页)
3. 核心指标 (1-2页)
4. 详细分析 (3-5页)
5. 问题与挑战 (1-2页)
6. 策略建议 (1-2页)
7. 下阶段计划 (1页)
8. 附录 (可选)
```

### 周报/月报 (8-12 页)
```
1. 封面
2. 本期概览 (1页)
3. 核心指标看板 (1-2页)
4. 分维度分析 (2-4页)
5. 亮点与问题 (1-2页)
6. 下周/月计划 (1页)
```

## 设计原则

### 内容原则
1. **结论先行** - 每页先说结论
2. **一页一观点** - 不要堆砌信息
3. **数据说话** - 用数字支撑观点
4. **视觉突出** - 大数字、清晰图表

### 色彩方案
```
主色：#1a365d (深蓝) - 专业、稳重
辅色：#3182ce (亮蓝) - 重点、强调
正向：#38a169 (绿色) - 增长、达标
负向：#e53e3e (红色) - 下降、风险
背景：#f7fafc (浅灰) - 干净、现代
```

### 字体规范
```
标题：28-36pt, 加粗
副标题：20-24pt
正文：16-20pt
数据：48-72pt (关键数字)
来源：12pt, 浅色
```

## 图表规范

### 图表选择
| 数据类型 | 推荐图表 |
|----------|----------|
| 趋势 | 折线图、面积图 |
| 对比 | 柱状图、条形图 |
| 占比 | 饼图、环形图 |
| 排名 | 条形图 |
| 漏斗 | 漏斗图 |

### 图表样式
```
- 标题清晰
- 坐标轴标签完整
- 数据标签适度
- 颜色一致
- 来源标注
```

## 使用 document-skills:pptx

调用现有 skill 生成 PPT：

```
使用 pptx skill 创建一份高管汇报 PPT：

标题：Q4 电商业务季度汇报

页面：
1. 封面 - 标题、时间、汇报人
2. 执行摘要 - 3个亮点、2个关注点
3. 核心指标看板 - GMV/订单量/客单价/DAU
4. GMV 趋势分析 - 折线图 + 解读
5. 渠道分析 - 饼图 + 洞察
6. 用户分析 - 漏斗图 + 建议
7. 策略建议 - 3个重点行动
8. 下季度计划 - 里程碑时间线

风格：商务、简洁、蓝色主题
```

## 文件命名规范

```
{类型}_{时间范围}_{日期}.pptx

示例：
- q4_report_202401.pptx
- weekly_report_20240115.pptx
- executive_summary_202401.pptx
- gmv_analysis_jan_20240201.pptx
```

## 演示技巧

### 开场
- 1分钟内说清核心结论
- 明确本次汇报的目的

### 中间
- 数据要有解读
- 每个发现要有建议
- 控制每页讲解时间

### 结尾
- 重申关键信息
- 明确下一步行动
- 留出 Q&A 时间
