# 高管报告 (Executive Report Workflow)

## 触发场景

- 季度/月度汇报
- 高管会议准备
- 董事会报告
- 战略决策支持

## 特点

- **简洁有力**: 每页不超过 3 个关键信息
- **结论先行**: 先说结论，再给数据支撑
- **视觉突出**: 大数字、清晰图表
- **行动导向**: 每个发现都有建议

## PPT 结构

### 标准高管报告结构 (10-15 页)

```
1. 封面
   - 报告标题
   - 时间范围
   - 汇报人/部门

2. 执行摘要 (1 页)
   - 3-5 个核心结论
   - 一句话总结

3. 核心指标概览 (1-2 页)
   - 关键指标仪表盘
   - 同比/环比
   - 目标达成率

4. 业绩分析 (3-4 页)
   - GMV/收入趋势
   - 利润分析
   - 增长驱动因素

5. 业务洞察 (2-3 页)
   - 关键发现
   - 市场动态
   - 竞品对比（如适用）

6. 问题与挑战 (1-2 页)
   - 当前痛点
   - 风险预警

7. 策略建议 (1-2 页)
   - 优先事项
   - 资源需求
   - 预期效果

8. 下阶段计划 (1 页)
   - 里程碑
   - 时间节点

9. 附录 (可选)
   - 详细数据表
   - 计算口径
```

## 视觉设计原则

### 色彩
```
主色: #1a365d (深蓝 - 专业感)
辅色: #3182ce (亮蓝 - 重点)
强调: #38a169 (绿 - 正向)
      #e53e3e (红 - 负向)
背景: #f7fafc (浅灰)
```

### 字体
```
标题: 28-36pt, 加粗
正文: 18-24pt
数据: 48-72pt (关键数字)
来源: 12pt, 浅色
```

### 布局
```
┌────────────────────────────────────┐
│  标题区 (10%)                      │
├────────────────────────────────────┤
│                                    │
│  内容区 (80%)                      │
│  - 左侧: 图表/数据                 │
│  - 右侧: 解读/要点                 │
│                                    │
├────────────────────────────────────┤
│  来源/备注 (10%)                   │
└────────────────────────────────────┘
```

## 内容模板

### 执行摘要页
```markdown
## 本期核心结论

### 📈 亮点
1. **GMV 同比增长 35%** - 首次突破 5000 万
2. **新客增长 28%** - 渠道投放效率提升
3. **毛利率提升 2pp** - 供应链优化见效

### ⚠️ 关注
1. 老客复购率下降 5pp - 需加强用户运营
2. 退货率上升 1.5pp - 商品质量需关注

### 🎯 下阶段重点
1. 启动会员体系升级
2. 优化退货处理流程
```

### 指标看板页
```markdown
## 核心指标看板

| 指标 | 本期 | 目标 | 达成率 | 同比 |
|------|------|------|--------|------|
| GMV | ¥5,200万 | ¥5,000万 | 104% | +35% |
| 订单量 | 42万 | 40万 | 105% | +28% |
| 客单价 | ¥124 | ¥125 | 99% | +5% |
| 毛利率 | 28% | 27% | 104% | +2pp |
```

### 洞察发现页
```markdown
## 关键发现：新渠道贡献超预期

### 数据支撑
- 抖音渠道 GMV 占比从 8% 提升至 18%
- 新渠道获客成本比传统渠道低 30%
- 新渠道用户 7 日留存率达 45%

### 业务解读
短视频平台用户与现有用户重叠度低，
有效拓展了新的增长空间。

### 建议行动
• Q3 加大抖音渠道投入 50%
• 组建专项运营团队
• 优化商品选品策略
```
